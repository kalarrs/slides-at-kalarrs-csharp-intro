General Confusion

Not everything is "Serverless"

However everyone will refer to it as "Serverless"

Let's Clarify

FaaS

Function as a service (FaaS) is a category of cloud computing services that provides a platform allowing customers to develop, run, and manage application functionalities without the complexity of building and maintaining the infrastructure.

AWS Lambda
Microsoft Functions
Google Cloud Functions

Serverless

Serverless computing is a cloud-computing execution model in which the cloud provider acts as the server, dynamically managing the allocation of machine resources.
Typically by utilizing a FaaS service provider.

Function invocation

Most serverless functions are invoked by events. Many common systems produce events. You can also invoke a serverless function on demand.

  • Api
  • File System Events (S3)
  • DB Events
  • Logging Events
  • Queues
  • Other Functions (Pipeline)
  • ...

#1 Creating functions in
AWS Console!

Create a function in under a minute! Just go to AWS Lambda Console

https://console.aws.amazon.com/lambda/home?region=us-east-1#/home

Demo

Pitfalls

  • Runtime languages are easy to get started with, but not compiled languages.
  • C# code can't be edited in the browser. This is because it is a jitted language.
  • Really no instructions on what everything means. You would have to read a lot of docs.
  • Not clear how to upload your own code.
