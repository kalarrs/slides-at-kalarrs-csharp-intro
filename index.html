Set of tooling built on top of Serverless Framework, mainly as plugins. As well as a collection of libraries to normalize development between languages.

Without Serverless Framework

  • Manually Create API Gateway
  • Manually Create API Gateway Resources
  • Manually Create and Upload Lambda Code
  • Manually Configure Environment variables
  • Wire events to Lambdas
  • Setup custom code backups
  • Multiple cloud providers is difficult

What does Serverless do?

  • Automates the infrastructure process
  • Automatically create cloud formations
    • Creates resources (All)
    • Configures Events
    • Configures Environment variables
    • Organizes code
  • Stores Code on s3
  • Deploys Code
  • Supports multiple cloud providers

What Serverless Framework doesn't do

  • Provide Architecture and Design patterns
  • Generate code
  • Visualize Infrastructure
  • Use latest code
  • Support best practices for hitrust/HIPPA compliance

What is a Project?

The code, documentation, & tests for a micro service. Typically an api resource, scheduled job, or part of a pipeline.
